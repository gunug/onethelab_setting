# Claude Code 도구 목록

Claude CLI에서 사용하는 도구들과 HTML 클라이언트에서의 추가 정보 표시 구현 상태.

## 구현 상태 범례

- ✅ **구현됨**: HTML 클라이언트에서 추가 정보 UI 표시
- ⚠️ **기본 표시**: 도구 호출만 표시 (추가 정보 없음)
- ❌ **미구현**: Python에서 파싱하지 않음

|------|--------|----------------|------|

## 구현된 추가 정보 UI

### Edit 도구 (✅)
- 변경 전/후 내용을 diff 형태로 표시
- 빨간색 배경: 변경 전 (old_string)
- 초록색 배경: 변경 후 (new_string)
- 접기/펼치기 기능

### Write 도구 (✅)
- 생성되는 파일 내용 표시
- 최대 500자까지 표시
- 접기/펼치기 기능

### Bash 도구 (✅)
- 실행되는 명령어 표시
- 최대 100자까지 표시
- 별도 스타일 블록 (어두운 배경)

### TodoWrite 도구 (✅)
- 할 일 목록 표시
- 상태별 아이콘:
  - ○ 대기 (pending)
  - ◐ 진행중 (in_progress)
  - ✓ 완료 (completed)
- 완료 항목에 취소선 스타일
- 접기/펼치기 기능

### AskUserQuestion 도구 (✅)
- 질문 텍스트 및 헤더 태그 표시
- 옵션 버튼 UI (라벨 + 설명)
- 단일 선택 / 멀티 선택 지원
- 기타(직접 입력) 옵션 지원
- 응답 제출 버튼 및 완료 상태 표시
- 응답을 채팅 메시지로 전송

## 향후 구현 가능한 기능

### Read 도구
- 읽은 파일의 줄 수 표시
- 파일 경로 전체 표시

### Grep 도구
- 검색 패턴 및 검색 경로 표시
- 검색 결과 개수 표시

### Task 도구
- 서브 에이전트 유형 표시
- 에이전트 실행 상태 표시

### WebFetch/WebSearch 도구
- URL 또는 검색 쿼리 표시
- 결과 미리보기

## 관련 파일

- Python 파싱: `chat_bot/chat_bot.py` (run_claude_stream, ask_claude 함수)
- HTML 표시: `chat_client/chat.js` (updateProgressToolStart 함수)
- CSS 스타일: `chat_client/index.html` (style 섹션)
